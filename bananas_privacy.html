Bananas Privacy Policy

Effective Date: April 15, 2025

Bananas is a trading behavior research and analytics system built and managed by its creator, intended for private or controlled-access use only during its development phase.

1. Data Collection

This GPT-based assistant may process inputs such as clone IDs, trading metadata, or session information if such data is shared during usage. No user-identifying personal data is collected intentionally.

2. Data Usage

All data is used strictly for the purpose of assisting with infrastructure development, code generation, system logging, and other backend engineering tasks for the Bananas project.

3. Data Storage

No data from this GPT is stored or used outside of the active session. Inputs and results are ephemeral unless manually logged or routed by the user to an external endpoint.

4. Third-Party Access

No user data is sold or shared with third parties. This assistant does not interact with advertising networks or analytics providers.

5. API Interaction

If API Actions are enabled, inputs may be sent to a private API endpoint hosted by the project owner. These endpoints do not store identifying data and exist solely for backend development.

6. User Responsibility

Users are responsible for avoiding the entry of sensitive or personal information. This system is not intended for public use or financial decision-making at this time.

7. Contact

For questions, reach out to the developer via the associated GPT profile or hosting platform.
